# Connor Pu - Personal Website

这是一个完整的个人网站，包含4个独立页面，展示了Connor Pu的作品、投资洞察和个人宣言。

## 文件结构

- `index.html` - **主页面（Main Page）** - 这是网站的入口
- `ai-apps.html` - AI应用和电影页面（Second Page）
- `investment-insight.html` - AI投资洞察页面（Third Page）
- `declaration.html` - 个人宣言页面（Fourth Page）

## 导航说明

### 主页面 (index.html)
- 这是网站的入口页面
- 点击右上角导航栏的按钮可以跳转到：
  - **AI APP & AI FILM** → `ai-apps.html`
  - **AI INVESTMENT INSIGHT** → `investment-insight.html`
  - **MY DECLARATION** → `declaration.html`
- 点击左上角的 **Connor Pu** 按钮可以刷新主页

### AI应用和电影页面 (ai-apps.html)
- 点击左上角的 **Connor Pu** 按钮返回主页 (`index.html`)
- 导航栏可以跳转到其他页面：
  - **AI App & Film** → 当前页面
  - **Investment Insight** → `investment-insight.html`
  - **Declaration** → `declaration.html`

### AI投资洞察页面 (investment-insight.html)
- 点击左上角的 **Connor Pu** 按钮返回主页 (`index.html`)
- 导航栏可以跳转到其他页面：
  - **AI App & Film** → `ai-apps.html`
  - **Investment Insight** → 当前页面
  - **Declaration** → `declaration.html`

### 个人宣言页面 (declaration.html)
- 点击左上角的 **WHO AM I** 按钮返回主页 (`index.html`)
- 点击左上角的 **Connor Pu** 按钮返回主页 (`index.html`)
- 导航栏可以跳转到其他页面：
  - **WHO AM I** → `index.html`
  - **AI APP & AI FILM** → `ai-apps.html`
  - **AI INVESTMENT INSIGHT** → `investment-insight.html`
  - **MY DECLARATION** → 当前页面

## 使用方法

1. **打开 `index.html` 作为网站入口**
2. 所有页面之间都可以通过导航栏互相跳转
3. 每个页面都是独立的HTML文件
4. 网站包含丰富的动画效果和交互功能

## 技术特性

- 响应式设计，支持移动端和桌面端
- 使用GSAP进行动画效果
- 自定义光标效果
- 平滑滚动
- 视差效果
- 3D倾斜卡片效果

## 注意事项

- 确保所有HTML文件在同一目录下
- 需要网络连接以加载外部资源（CDN）
- 建议使用现代浏览器（Chrome, Firefox, Safari, Edge）

## 访问方式

直接在浏览器中打开 `index.html` 文件即可开始浏览网站。
